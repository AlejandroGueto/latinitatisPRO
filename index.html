Soporte de Voz

Esta web utiliza el motor de voz de su sistema. Si el griego no suena o lee "punto", debe instalar el sintetizador:

• Windows: Configuración > Hora e idioma > Voz > Agregar voces (Griego).
• Android/iOS: Accesibilidad > Contenido leído > Voces.

El Oráculo de Sabiduría

Requiere una API Key de Groq gratuita para realizar análisis morfosintácticos y etimológicos profundos. Sus datos permanecen privados en su navegador.

1. Inicie sesión en Groq Console.

2. Pulse "Create API Key" y copie el código gsk_...

3. Péguelo cuando seleccione una palabra en el lector.
